





# 软件介绍

`cordova-plugin-wechat` 是基于微信（OpenSDK）开发的Cordova插件。他实现官方SDK的所有接口，并在参数和功能性进行扩展，方便你快速在cordova及相关程序上快速进行微信登录，支付，分享等功能的接入和开发。



[主页](https://www.byteee.fund/cordova-plugin-wechat)

[文档](https://www.byteee.fund/cordova-plugin-wechat/)



## SDK版本

- Cordova 12.x
- 微信 OpenSDK Android 6.8.24，  IOS 2.0.2



## 功能列表

- [x] 微信登录 （授权登录，扫码登录）

- [x] 微信支付   

- [x] 微信分享/收藏 (文本，图片，网页，视频，音乐，音乐视频，文件，表情，小程序)

- [x] 拉起小程序

- [x] 拉起微信客服

- [x] 一次性订阅消息

- [x] 发票

- [x] 检查微信APP是否安装
